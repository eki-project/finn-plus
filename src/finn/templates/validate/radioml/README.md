# RadioML 2018.01A Dataset

The RadioML 2018.01A dataset is a synthetic dataset for radio signal modulation recognition. It contains 24 different modulation types and various signal-to-noise ratios (SNRs).

## Download Location

The dataset was created by deepsig.ai but their [download link](https://www.deepsig.ai/datasets/) seems to be down. But it can be found on [Kaggle](https://www.kaggle.com/datasets/pinxau1000/radioml2018),
